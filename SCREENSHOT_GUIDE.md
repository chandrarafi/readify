# Screenshot Guide untuk Petunjuk

## Folder

Simpan semua screenshot di: `assets/petunjuk/`

## Screenshot yang Dibutuhkan

### 1. belajar.png

- **Capture dari**: Menu Belajar (Alfabet atau Kosakata screen)
- **Fokus**: Tampilkan huruf/kata dengan tombol navigasi
- **Ukuran**: Landscape, 800x600px atau sesuai aspect ratio device
- **Tips**: Ambil saat menampilkan huruf A atau kata "buku"

### 2. speaker.png

- **Capture dari**: Screen yang ada tombol speaker (Kosakata screen)
- **Fokus**: Highlight tombol speaker oranye
- **Ukuran**: Landscape, 800x600px
- **Tips**: Zoom in sedikit ke area tombol speaker

### 3. latihan.png

- **Capture dari**: Tebak Kata screen
- **Fokus**: Tampilkan soal dengan 3 pilihan jawaban
- **Ukuran**: Landscape, 800x600px
- **Tips**: Ambil saat soal pertama muncul

### 4. histori.png

- **Capture dari**: Histori screen
- **Fokus**: Tampilkan list nilai dengan emoji dan warna
- **Ukuran**: Landscape, 800x600px
- **Tips**: Pastikan ada beberapa nilai tersimpan

### 5. musik.png

- **Capture dari**: Menu screen
- **Fokus**: Highlight tombol musik di pojok
- **Ukuran**: Landscape, 800x600px
- **Tips**: Bisa edit dengan arrow/circle untuk highlight tombol

### 6. navigasi.png

- **Capture dari**: Screen manapun yang ada tombol back
- **Fokus**: Highlight tombol panah kiri atas
- **Ukuran**: Landscape, 800x600px
- **Tips**: Bisa edit dengan arrow/circle untuk highlight tombol

## Cara Ambil Screenshot

### Di Android:

1. Buka aplikasi Readify
2. Navigate ke screen yang ingin di-capture
3. Tekan **Volume Down + Power** bersamaan
4. Screenshot tersimpan di Gallery

### Di Emulator:

1. Buka aplikasi di emulator
2. Klik icon **Camera** di toolbar emulator
3. Atau tekan **Ctrl + S** (Windows) / **Cmd + S** (Mac)

## Edit Screenshot (Optional)

Gunakan tools seperti:

- **Paint** (Windows) - untuk crop dan resize
- **Preview** (Mac) - untuk crop dan resize
- **Canva** - untuk tambah arrow/highlight
- **Figma** - untuk edit profesional

## Tips:

- Pastikan screenshot jelas dan tidak blur
- Crop bagian yang tidak perlu (status bar, dll)
- Resize ke ukuran yang konsisten
- Compress jika file terlalu besar (gunakan TinyPNG)

## Fallback

Jika screenshot belum ada, aplikasi akan menampilkan placeholder icon. Tapi lebih baik pakai screenshot asli untuk pengalaman yang lebih baik!
